# Real-Time Chat Application

## About
A real-time web based chat application built with Flask that supports multiple chat rooms and user authentication. This project focusses on learning full-stack web development fundamentals, particularly database design and real-time communication.

## Tech Stack
**Backend:** Python, Flask, Flask-SocketIO, Flask-Login

**Database:** SQLite with raw SQL queries

**Frontend:** HTML, CSS, Vanilla JavaScript

**Real-Time:** WebSocket (Socket.IO)
